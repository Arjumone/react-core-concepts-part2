# React Users Data Project

## 📌 Project Overview
This is a simple React.js project where I practiced using different React hooks and fetching data from an API. The project displays a list of users by retrieving data from an external source.

## 🚀 Technologies Used
- React.js
- JavaScript (ES6)
- JSONPlaceholder API

## ⚙️ Features
- Fetch user data from an API
- Display users dynamically on the webpage
- Practice using React hooks such as:
  - useState

## 🧠 What I Learned
Through this project I learned:
- How to use **useState** for state management
- How to render dynamic data in React
